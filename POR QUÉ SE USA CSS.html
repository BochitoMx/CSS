¿POR QUÉ SE USA CSS?

El uso de CSS responde a la necesidad de aplicar el principio de separación de presentación y contenido, lo cual resulta esencial para una buena práctica en el desarrollo web. La idea es que los documentos HTML se enfoquen exclusivamente en el contenido, como texto, imágenes y datos que se quieren trasmitir, mientras que los documentos CSS gestionen únicamente el estilo y la apariencia, como colores, tipografías, diseños y formas.

Esta separación trae consigo varias ventajas importantes:

  • Facilidad de modicación

    • Reducción de la duplicación de código:

  • Al tener los estilos centralizados, se evita la repetición de código en diferentes partes del sitio. Esto facilita la organización del código y permite hacer cambios de manera más eficiente.

    • Optimización del rendimiento

  • Al separar el contenido de la presentación, los archivos HTML se vuelven más ligeros, lo que disminuye la cantidad de información que debe descargarse, acelerando la carga de las páginas web.

    • Adaptabilidad a diferentes dispositivos

  • El uso de CSS facilita la creación de versiones del sitio optimizadas para diferentes dispositivos, como tablets y smartphones, mediante el uso de técnicas como las media querías. Esto mejora la experiencia del usuario en una variedad de pantallas y resoluciones.
